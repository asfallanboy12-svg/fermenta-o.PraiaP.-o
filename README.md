
Praia Pão — Controle de Fermentação (Demo)

Como publicar no Vercel (rápido):
1. Crie conta em https://vercel.com e faça login.
2. Suba este código para um repositório no GitHub (ou use o Vercel CLI).
3. No Vercel: New Project -> Import Git repository -> escolha o repositório.
4. Build command: npm run build
5. Output directory: build
6. Deploy. Se escolher nome 'praiapao-fermentacao' e estiver disponível, será usado como subdomínio.
